# 👤 Add Friend – Toggle Button UI

A clean and interactive profile card built using **HTML, CSS & JavaScript**.  
This project demonstrates DOM manipulation, toggle functionality, and attention to UI/UX details — with a smooth, responsive card design.

## ✨ Project Highlights

- ✅ Built with pure **HTML5**, **CSS3**, and **Vanilla JavaScript**
- 🔁 **Toggle button** that switches between `Add Friend` and `Friends ✓`
- 🎨 Clean, modern **profile card UI**
- 💨 Smooth transitions and hover effects
- 🧠 Follows **modern UI/UX design principles**
- 💡 Proper use of **semantic HTML** and structured CSS
- 🛠️ Optimized for readability and maintainability

## 📌 How It Works

- By default, the button shows **"Add Friend"** and status shows **"Stranger"**
- On click → button changes to **"Friends ✓"** and status updates to **"Friend"**
- Clicking again → toggles back to the original state
